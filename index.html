CANNY

For the Canny I chose to use the thresholds 100,450 because I believed that it gave me a good indicator of strong lines. I checked with different types and when I used thresholds like 800,1000 there were very little lines. When I used 40,100, I was getting a lot of lines. So when it had the smaller thresholds it was noticing everything. So I chose to have 100,150 because I only wanted to show the really strong edges. I did my testing with an image of the plants and window; my threshold was about to see the edge of the window, the plant against the sky and the door on the right. (I put some small observations above certain pictures). But the final choice of threshold I used was 100,150 because I believe it gave the best result.

I used the site we were directed from, opencv.org in order to learn about this. I chose to use the same threshold for my Canny. Some of the images did not provide any lines from the Canny edge detection. When the thresholds were. PROBALISTIC HOUGH

The biggest difference that I saw was that there more lines. But they were also a lot smaller. So it was getting more lines, because it was looking for smaller lines. Or maybe it was because the standard uses more of the angles on the polar coordiantes. The probalistic Hough is only finding the lines which line on the either the X or the Y axis. Those are the differences. So each one has a strength.
